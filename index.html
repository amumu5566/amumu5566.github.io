[Begining C++] Default Constructor

重點1 :
所謂default constructor, 是在沒有參數的情況下呼叫的constructor
如果沒有定義, compiler會為你產生預設的預設建構子(default default constructor)

重點2 : 自己定義甚麼事都不做的default constructor
跟compiler提供的有微妙的差別!!

重點3 : 如果有自己定義constructor了, 那compiler就不會再提供default default constructor, 所以要自己定義default constructor

[Beginning C++] Define Outside

在類別之外定義member function

  • 把類別定義放在標頭檔(header file)
  • 把member function的定義放在對應的原始檔

可以使原始碼更容易管理

  • 任何使用Box類別的原始檔只要include標頭檔即可
  • 如果更改member function的實作, 只要類別定義不變, 就不影響該類別程式的操作
